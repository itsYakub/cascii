DESCRIPTION:

	"cascii" is a command-line utility for displaying standard ascii table,
	from 0 to 127, written in C without any dependency.

===================================

BUILD AND SETUP:

	0. Cloning repository from GitHub:

		$ git clone git@github.com:itsYakub/cascii.git
		$ cd cascii

	1. Build a project:
		
		$ ./BUILD.sh

	2. Install the executable:
		
		$ ./INSTALL.sh

	3. Cleaning and uninstalling:

		$ ./CLEAN
		$ ./UNINSTALL
